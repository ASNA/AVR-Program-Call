PgmCall is an ILE RPG program used by AVR to demonstrate
its IBM i program call feature with the Call/Parm syntax.

The full ASNA Visual RPG .NET program to call this program
is here:
https://github.com/ASNA/AVR-Program-Call

To compile PgmCall:
Add RPMRDATA and RPTEST then use CRTBNDPGM.

At ASNA, PgmCall is on Cypress in the RPTest library.
